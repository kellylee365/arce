standards compliant min-width

{width:90%; min-width:400px;} includes IE5+ and IE6

this is the standards compliant way that I have found to emulate true min-width in IE which just requires an extra outer container.

this div has a min-width of 400px and a width of 90%. You can set the width as any percentage and the min-width as a px or em value.

discovered 19th May 2005 by Stu Nicholls

this div is 400px wide and can be used as a guide for min-width.
