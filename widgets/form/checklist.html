Checklists

An alternative to the select element with enabled multiple attribute.

This example is described in the Check it, don't select it article on C82. See a bare-bones example.

Compatibility:

  • Win: IE 5.0+, Firefox 0.8+
  • Mac: Safari 1.2+, IE 5.2, Camino 8.2+, Netscape 7+, OmniWeb 4.1+, iCab 3.0+

Feel free to use the example on this page in your own works. If you choose to do so, drop me a line. I'd like to hear how you used it.
